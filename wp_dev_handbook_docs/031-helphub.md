What is HelpHub?
----------------

HelpHub is the ultimate home for all of the WordPress user documentation previously resided in the [WordPress Codex](https://codex.wordpress.org/). ‘HelpHub’ is the project name and part of [WordPress.org Documentation](https://wordpress.org/documentation/).

For more information see [the original announcement](https://make.wordpress.org/support/2015/01/kicking-off-helphub/) as well as the [full project spec](https://make.wordpress.org/meta/2015/01/13/help-hub-spec/), or all posts regarding HelpHub can be found [here](https://make.wordpress.org/docs/category/helphub/).
